### 2. EFFETS LUMINEUX ET GLOW (NÉON DOUX)

**Objectif** : Tous les éléments lumineux (nœuds, filaments, UI) doivent avoir un glow doux et organique

**Implémentation** :
- Créer un composant `<GlowEffect>` réutilisable avec Skia
- Technique : multi-pass blur avec additive blending
  - 1er pass: élément de base
  - 2e pass: blur gaussien large (radius 10-15)
  - 3e pass: blur gaussien étroit (radius 5-8)
  - Combiner avec mode `screen` ou `add`
- Animation du glow : pulsation douce (scale 0.95 → 1.05, durée 2s, easing sine)
- Intensité variable selon l'état : hover +30%, selected +50%, combo +100%

**Shader GLSL custom pour glow organique** :
```glsl
// glow.frag
uniform vec2 resolution;
uniform float time;
uniform vec3 color;
uniform float intensity;

void main() {
  vec2 uv = gl_FragCoord.xy / resolution;
  float dist = length(uv - 0.5);
  
  // Glow principal
  float glow1 = 1.0 / (dist * 10.0 + 1.0);
  // Glow secondaire plus large
  float glow2 = 1.0 / (dist * 5.0 + 1.0) * 0.3;
  
  // Pulsation organique
  float pulse = sin(time * 2.0) * 0.1 + 0.9;
  
  vec3 finalGlow = color * (glow1 + glow2) * intensity * pulse;
  gl_FragColor = vec4(finalGlow, glow1);
}
```

### 6. SYSTÈME AUDIO PROCÉDURAL

**Objectif** : Musique générative qui réagit au gameplay + sons cristallins synchronisés

**MusicEngine.ts avec Tone.js** :
- Synthétiseur polyphonique : Tone.PolySynth avec envelope ADSR doux
- Gamme pentatonique par monde :
  - Monde 1 : C Major Pentatonic [C, D, E, G, A]
  - Monde 2 : A Minor Pentatonic [A, C, D, E, G]
  - Monde 3 : E Minor Pentatonic [E, G, A, B, D]
  - etc.
- Mapping couleur → note :
  - Rouge = Do (C4)
  - Orange = Ré (D4)
  - Jaune = Mi (E4)
  - Vert = Sol (G4)
  - Bleu = La (A4)
  - Violet = Si (B4)
- Boucle ambiante : Tone.Sequence avec patterns aléatoires (Markov chain)
- Réverbération : Tone.Reverb (decay 2-3s, wet 0.3)
- Filtre passe-bas : Tone.Filter pour adoucir

**SoundManager.ts avec expo-av** :
- Pre-load tous les sons au démarrage de l'app
- Sons cristallins : samples de cloche/carillon/glass harmonics
- Sons d'erreur : notes graves douces (jamais harsh)
- Sons de victoire : crescendo de harpe ascendante
- Pool de sons (reuse instances pour performance)

```typescript
class ParticlePool {
  private pool: Particle[] = [];
  private active: Particle[] = [];
  
  get(): Particle {
    return this.pool.pop() || new Particle();

  release(particle: Particle): void {
    particle.reset();
    this.pool.push(particle);
  }
```

**Nebula Shader** pour arrière-plans :
```glsl
// nebula.frag - Bruit fractal pour nébuleuses
uniform float time;
uniform vec2 resolution;

// Fonction noise Perlin 3D
float noise(vec3 p) {
  // Implementation Perlin noise
}

void main() {
  vec2 uv = gl_FragCoord.xy / resolution;
  
  // Multi-octave noise
  float n = noise(vec3(uv * 3.0, time * 0.1));
  n += 0.5 * noise(vec3(uv * 6.0, time * 0.15));
  n += 0.25 * noise(vec3(uv * 12.0, time * 0.2));
  
  // Couleur basée sur le monde
  vec3 color1 = vec3(1.0, 0.6, 0.8); // Rose
  vec3 color2 = vec3(0.6, 0.4, 1.0); // Violet
  vec3 finalColor = mix(color1, color2, n);
  
  gl_FragColor = vec4(finalColor, 0.8);
}
```
